Please develop an Androd application to monitor tyre pressures on a car.

## Use cases

### Use case 1:
User can enter the type pressure for a tyre after measuring. 
User can optionally enter the mileage
User can optionally enter the pressure after inflating the tyre


### Use case 2:

User can visualise the rate of tyre deflation to identify tyre faults
User can visualise the rate either by miles or by calendar time. 


## UI considerations

A simple UI with 4 buttons representing the 4 tyres on a car


## Tech considerations

- Develop a simple app devised for personal use on a fairly wide range of phones
- Only use android features available up till 2022.
- It doesn't need to support iOS

## Developent considerations
- Author is novice at Androd dev so include plenty of code comments
- Assume the project will be published as open source project so include README, break features into commits 

## Security

- don't make network calls outside of the app.
- it should store data locally on the phone.
